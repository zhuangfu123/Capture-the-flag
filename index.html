Welcome to Capture the flag!

In this game, you compete with others to solve challenges and find hidden flags. The game will be played entirely in the terminal, so be prepared to use your coding skills. Here's how to get started:

Step 1: Make sure you have these tools on your computer, we've downloaded them for you

  • Terminal emulator (such as terminal on Mac or command prompt on Windows)
  • Python 3.x
  • Git

If you find that your computer does not have these tools, please call the teacher.

Step 2: Clone repository Game challenges and tokens are stored in Git repositories. First, clone the repository to the local machine using the following command:

git clone [repository URL]

Replace [repository URL] with the URL of the repository provided by the game administrator. Once you've cloned the repository, you're ready to play.

Step 3: Solve the challenge.

This flag will be a file in the Linux file system that you must find, and grant read permission to. Once the file has been read, the CTF will be considered complete. A cheat sheet and instruction sheet will be provided.

Are you ready? Join the game and feel the charm of computer science!

Attention! You only have 10 minutes.
